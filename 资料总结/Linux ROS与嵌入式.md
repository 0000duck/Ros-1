# 3.用ASIO读写设备串行口
     ASIO不仅支持网络通信还能支持串口通信。如何让两个设备使用串口通讯，最重要的是设置好正确的参数，
     那么串口的参数就包括：
     波特率、
     奇偶校验位、
     停止位、
     字符大小和
     流量控制，
     两个设备只有设置相同的参数才能相互间进行数据交换。
